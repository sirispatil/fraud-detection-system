# Hybrid Session-Level Fraud Detection System

## Description
This project detects fraud in e-commerce transactions using:
- Rule-based detection
- Machine Learning model
- Risk escalation logic

## Tech Stack
- MySQL
- Flask (Backend)
- Streamlit (Frontend)
- Python ML model

## Features
- Real-time fraud detection
- Admin dashboard
- Risk scoring system

## How to Run
1. Run SQL file in MySQL
2. Start backend (Flask)
3. Start frontend (Streamlit)
